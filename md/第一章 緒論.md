# 第一章　緒論

---

## 1.1 研究背景與動機

近年來，人工智慧（Artificial Intelligence, AI）技術的快速發展，使強化學習（Reinforcement Learning, RL）從研究領域逐漸進入大眾視野。AlphaGo 擊敗人類圍棋冠軍、自動駕駛車輛在複雜路況中習得駕駛策略、機器人透過反覆嘗試學會精確操作——這些應用的背後，皆以強化學習作為核心機制。隨著 AI 素養教育的推動，如何讓學習者理解強化學習的基本原理，成為當前教育領域值得探討的課題。

然而，強化學習的教學面臨一個根本困難：**它的學習歷程高度抽象，且難以直接觀察**。學習者必須同時理解「狀態、動作、獎勵」的循環概念、Q 值的累積與更新、探索與利用的取捨，以及策略收斂的過程——這些概念彼此關聯、動態變化，若僅透過數學公式或靜態圖示說明，初學者很難建立直觀的理解。

現有的強化學習教學工具各有侷限。Gymnasium（原 OpenAI Gym）是研究社群的標準平台，提供豐富的環境與完整的 API，但需要具備一定程式能力才能操作，對初學者而言學習曲線陡峭。圖形化積木程式設計工具（如 Blockly）雖大幅降低程式門檻，卻缺乏對強化學習概念的直接支援。介於兩者之間，目前缺乏一個既能讓學習者不需撰寫程式，又能直觀觀察強化學習全過程的教學平台。

基於上述背景，本研究設計並開發了 **Rein Room（強化教室，簡稱 RR）**——一套以使用者介面驅動（UI-driven）為核心設計理念的強化學習教學平台。Rein Room 讓學習者透過滑桿調整超參數、即時觀察訓練曲線、解讀 Q-table 熱力圖，在不接觸任何程式碼的情況下，完整經歷強化學習的試錯、更新與收斂過程。同時，平台亦設計了 Q-Table 蒸餾式深度 Q 網路（DQN）機制，並提供「知識同步率」視覺化指標，使學習者得以觀察神經網路如何從 Q-Table 中習得泛化能力，進一步深化對演算法本質的理解。

## 1.2 研究目的與研究問題

本研究的核心目的有二：其一，設計一套以降低 RL 抽象性為核心訴求的教學平台；其二，透過對照實驗評估該平台相較於 Gymnasium + Colab 環境在大學生與研究生族群中的教學成效。

據此，本研究提出以下三個研究問題：

**RQ1：概念理解**
使用 Rein Room 進行學習的學習者，在強化學習核心概念（狀態、動作、獎勵、Q 值、探索與利用等）的理解程度上，是否優於使用 Gymnasium + Colab 的學習者？

**RQ2：策略解讀**
在統一的圖表判讀測驗中，兩組學習者對訓練曲線與策略收斂的解讀能力是否存在差異？此 RQ 以紙本統一題目施測，排除平台介面熟悉度的影響，測量底層概念理解的深度。

**RQ3：學習動機與易用性**
使用 Rein Room 的學習者，在學習興趣、自我效能感與平台易用性的評估上，與使用 Gymnasium + Colab 的學習者相比，呈現何種差異？

## 1.3 研究範圍與限制

**研究範圍**

本研究以大學生與研究生為實驗對象，採準實驗設計（quasi-experimental design），以班級為單位分為實驗組（Rein Room）與對照組（Gymnasium + Colab），進行為期四節課的教學介入，並以前後測與問卷進行成效評量。

**研究限制**

1. **樣本侷限**：本研究以特定課程的修課學生為樣本，非隨機抽樣，結果的外部效度有限，難以直接推論至其他族群。

2. **圖表判讀比較的基礎差異**：RQ2 採用統一紙本題目以控制平台介面差異，但兩組學習者在學習過程中接觸到的視覺化工具性質不同（互動式 vs. 靜態程式輸出），仍可能對解讀能力產生影響。

3. **平台功能邊界**：Rein Room 目前以 Q-Learning 與蒸餾式 DQN 為核心，暫未涵蓋 Policy Gradient 等進階演算法，教學深度以 RL 入門概念為主。

4. **純前端架構**：平台以純前端技術實作，無後端資料庫，學習者的操作行為紀錄依賴瀏覽器端，資料蒐集方式有其限制。

## 1.4 論文架構說明

本論文共分六章，各章內容簡述如下：

**第一章 緒論**
說明研究背景與動機、研究目的與問題、研究範圍與限制，以及論文整體架構。

**第二章 文獻探討**
回顧強化學習的教育潛力、現有教學案例與工具（實體機器人、虛擬平台、Gymnasium），以及 UI-driven 設計在低門檻 RL 教學中的應用，最終定位本研究的創新切入點。

**第三章 系統與教材設計**
介紹 Rein Room 平台的架構設計（前端介面、通訊協定、視覺化模組）、降低 RL 抽象性的視覺化設計理念（包含知識同步指標），以及配套教材的設計邏輯。

**第四章 教學實驗設計**
說明研究假設、A/B 實驗流程、實驗對象與分組方式、教學流程、評估工具設計，以及數據分析方法。

**第五章 實驗結果與分析**
依序呈現 RQ1–RQ3 的分析結果：概念理解測驗、策略解讀測驗、學習動機與易用性問卷，並輔以錯誤行為觀察與平台改進討論。

**第六章 結論與展望**
總結研究成果，討論平台的教育應用潛力，並提出未來擴充方向。
