## **5.1 RL 概念理解測驗分析 **

本研究以「RL 概念理解測驗」作為核心量化成效指標，透過課前（Pre-test）與課後（Post-test）兩次測驗，評估受試者在強化學習基礎概念上的理解提升程度，並比較不同教學平台與教材介入（實驗組：RL Lab；對照組：Gymnasium + Colab）對概念學習的影響。測驗題目設計聚焦於初學者最關鍵且能跨平台通用的概念，包括：state / action / reward（SAR）三要素、episode 的意義、探索與利用（exploration vs exploitation）的權衡、長短期回報直覺（折扣概念）、Q 值（state-action 價值）之基本理解，以及訓練曲線（reward 隨 episode 變化）之基礎判讀能力。由於本研究旨在比較不同教學媒介的教學支架與操作門檻，而非比較題目難度或高階數學推導，因此採用入門層次之題項以提高可作答性與測量穩定性。

### **5.1.1 測驗設計與評分方式**

RL 概念理解測驗共計 **[待填：15]** 題，皆為單選題，每題答對得 1 分、答錯得 0 分，總分範圍為 0～**[待填：15]** 分。為降低題目之外的干擾因素（如寫作能力差異），本研究以客觀計分為主，並使用相同題庫（或同構題庫）作為前後測工具，使前後測結果能反映受試者在同一概念集合上的變化。前後測配對以匿名 StudentID 進行，確保資料可追蹤且符合隱私保護需求。

在資料處理上，本研究以「有效配對樣本」作為主要分析對象，即受試者必須同時完成前測與後測，方納入前後比較；若僅完成前測或僅完成後測，則排除於前後差異分析之外，但可在描述統計階段以補充方式呈現。此做法可避免因缺失資料造成增益估計偏誤，並使組間比較更具可比性。最後，有效樣本數如下：實驗組 N = **[待填]**，對照組 N = **[待填]**，排除樣本原因包括 **[待填：缺後測/重複作答/StudentID 無法配對]** 等。

為確保量測工具具一定一致性，本研究亦可進一步以內部一致性指標評估測驗穩定度。由於本測驗採 0/1 二分計分，理論上可採 KR-20 或 Cronbach’s α 進行檢核；惟本研究之測驗設計以多概念面向為主，題目可能呈現多維度特性，因此一致性指標僅作為輔助參考，仍以「前後差異」與「組間比較」作為主要結論依據。

### **5.1.2 分析策略與統計方法**

本節分析目的可分為三個層次：（1）確認兩組在介入前的起點是否相近；（2）檢驗各組在介入後是否提升；（3）比較兩組提升幅度是否存在差異。為對應上述目的，本研究採用描述統計與推論統計搭配呈現。

首先，以描述統計呈現兩組在前測、後測與增益（gain）的平均數（M）與標準差（SD），並搭配箱型圖或分佈圖呈現分數分佈狀況。視覺化呈現可協助檢視極端值、分佈偏態與組間差異趨勢，也能為後續統計檢定的選擇提供依據。

其次，在推論統計方面，本研究採以下原則選擇檢定方法：若分數分佈近似常態且樣本數足夠，則採用參數檢定（t-test）；若分佈明顯偏態或樣本數偏小，則採用非參數檢定（Mann–Whitney U、Wilcoxon）。此外，單以 p 值判斷可能受樣本數影響，因此本研究同步報告效果量（effect size），例如獨立樣本比較之 Cohen’s d 或 r，與 ANCOVA 之 partial η²，以提升結果解釋力。

為確保實驗設計具可比性，本研究首先比較兩組受試者在前測分數上的差異，以檢驗介入前起點是否一致。若兩組前測無顯著差異，則後續可直接比較增益或後測；若兩組前測存在顯著差異，則需使用控制起點差異的方法（如 ANCOVA），以避免起點不一致導致組間結論偏誤。

### **5.1.4 組內前後比較（各組是否學到）**

在確認起點後，本研究針對兩組分別檢驗課程介入是否帶來概念理解提升。此分析以受試者前後測配對資料進行，若分佈符合假設採成對樣本 t 檢定；不符合則採 Wilcoxon 符號等級檢定。此部分的意義在於：即使組間差異不顯著，也可能出現「兩組都顯著進步」的情況，代表兩種教學方式皆能有效促進概念學習；反之若只有某一組顯著進步，則可能意味該平台介入提供較強的學習支架或較低的操作負擔。

### **5.1.5 組間成效比較（增益比較與控制起點）**

為比較兩種教學介入的相對成效，本研究計算每位受試者的增益（Gain），定義如下： **Gain = Post-test − Pre-test**。 增益作為個別學習提升量，能將先備差異部分抵銷，使組間比較更貼近教學介入效果。兩組增益可使用獨立樣本 t 檢定（或 Mann–Whitney U 檢定）比較其平均差異。

為使結果更直觀，本研究以圖表輔助呈現兩組前後測分佈與增益比較。建議至少包含以下圖表： （1）前測與後測分數分佈箱型圖：呈現中位數、四分位距與可能的離群值，便於觀察分佈差異與提升趨勢。 （2）增益比較圖（長條圖或箱型圖）：直接呈現兩組提升幅度差異。 （3）（可選）題項正確率比較圖：呈現最容易提升與最難改善的概念題型。
